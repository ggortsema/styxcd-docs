# Archive

This directory preserves historical notes, session handoffs, and superseded drafts.

The archive is intentionally retained for traceability, but it is not the primary reading path.

Use the main directories first:

- `adr/`
- `architecture/`
- `specifications/`
- `platform-targets/`
- `observability/`

When a historical note conflicts with an ADR or consolidated architecture document, the ADR/consolidated document wins unless a newer ADR says otherwise.
