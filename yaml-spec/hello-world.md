# Hello World

This is the smallest valid StyxCD workflow.

```yaml
workflow: cloud_workflow

release:
  name: hello-world
  version: 0.1.0
```

## Description

| Field | Description |
|---------|-------------|
| workflow | Workflow type |
| release.name | Release name |
| release.version | Release version |
